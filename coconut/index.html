Coconut

"Got it all here in my coconut!" — Claude

Persistent memory and task orchestration for AI agents.
Never lose context between AI sessions again.

View on GitHub Documentation

Features

🧠 Persistent Memory

Maintains project state across AI sessions so you never lose context

🎖️ Military-Style SITREPs

Structured situation reports optimized for AI communication

📋 Task Management

Track project progress with visual indicators and completion stats

🔄 Context Sync

Automatically collect and organize project context from git, database, and environment

🚨 Recovery System

Resume work seamlessly after crashes or interruptions

🤖 AI Agnostic

Works with Claude, ChatGPT, Cursor, and any AI that reads files

# Install Coconut
git clone https://github.com/jasonacurry/coconut.git
cd your-project
../coconut/install.sh

# Run Coconut
./scripts/coconut.sh
